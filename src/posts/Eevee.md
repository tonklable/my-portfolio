## Description
Eevee project is a part of Prototyping UX class lectured by Prof. Katie Seaborn, Tokyo Institute of Technology. This project is a Wizard of Oz prototype using **HTML/CSS/Javascript** with **Anime.js** to create fanart animation of Eevee from Pok√©mon. Eevee can attack, dodge and cry. It is connected to the backend by Node.js, Express and Socket.io.
